# Winception v2.0.0-alpha.6 candidate notes

Release channel: **unpublished internal test candidate**.

Alpha 6 preserves the accepted alpha.5 PXE and Software Test behavior and fixes installed diagnostics/evidence export. A packaged host no longer fails diagnostics only because global npm is absent. PowerShell module probes suppress import warnings before parsing JSON, manual full diagnostics select the newest Fleet run, and compressed bundles enumerate literal source entries so the ZIP must exist before the operation succeeds.

The candidate uses MSI `ProductVersion 2.0.21`, the existing explicitly trusted self-signed test certificate, and `AGPL-3.0-only`. It must pass local source/package gates and installed upgrade plus diagnostics bundle download before publication. Alpha.5 remains immutable and available as the last published internal prerelease.
