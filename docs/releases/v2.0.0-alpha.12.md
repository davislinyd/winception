# Winception 2.0.0-alpha.12 internal candidate

The WinPE torrent seed window now calculates its completion time, deadline and countdowns in UTC. This prevents Hyper-V time synchronization or time-zone changes from extending a 15-minute seed window by hours after image apply.

The candidate uses MSI `ProductVersion 2.0.27`, `AGPL-3.0-only`, and the explicitly trusted self-signed test certificate. Local source/package gates and two consecutive clean two-client PXE rounds are required; the reduced client count reflects the outer host memory limit and does not replace future physical-laptop acceptance.
