# Winception 2.0.0-alpha.12 internal candidate

This candidate is rejected and unpublished. Live PXE validation proved the synchronized WinPE UTC clock could still be offset by 16 hours, so an absolute UTC deadline was not a safe elapsed-time source.

The candidate used MSI `ProductVersion 2.0.27`, `AGPL-3.0-only`, and the explicitly trusted self-signed test certificate. Alpha.13 replaces the wall-clock deadline with a monotonic timer.
