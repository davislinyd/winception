# Winception 2.0.0-alpha.13 internal candidate

This candidate is rejected and unpublished. Two live rounds proved the monotonic seed timer, but the second round exposed a transient `File.Replace` sharing violation while the progress viewer read the JSON file. One client completed apps without reaching the final desktop-ready gate.

The candidate used MSI `ProductVersion 2.0.28`, `AGPL-3.0-only`, and the explicitly trusted self-signed test certificate. Alpha.14 retains the timer fix and retries transient atomic replacement sharing violations.
