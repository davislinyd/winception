# Winception 2.0.0-alpha.13 internal candidate

The WinPE torrent seed window now measures elapsed time with `System.Diagnostics.Stopwatch`, independent of wall-clock, time-zone and Hyper-V synchronization changes. Telemetry reports monotonic seconds remaining; the server derives the operator-visible deadline from its own clock.

The candidate uses MSI `ProductVersion 2.0.28`, `AGPL-3.0-only`, and the explicitly trusted self-signed test certificate. Local source/package gates and two consecutive clean two-client PXE rounds are required; the reduced client count reflects the outer host memory limit and does not replace future physical-laptop acceptance.
