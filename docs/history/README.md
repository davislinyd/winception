# Historical manuals

`docs/winception-operations-manual.html` is retained at its legacy URL for v1 compatibility and historical evidence. It is not a v2 source of truth and must not receive parallel v2 edits.

Winception v2 documentation is authored only in `apps/docs/docs` and its English mirror under `apps/docs/i18n/en`. The build produces GitHub Pages `/winception/` and the installed offline `/manual/` from that same MDX source.
