# Release Status, Support and License

Until elevated fresh-VM lifecycle, single-client PXE and Software Test acceptance are complete, v2 remains an internal prerelease and is not production-ready.

## Release assets

A prerelease contains the MSI, CER, `Install-Winception.ps1`, release manifest, `SHA256SUMS`, CycloneDX SBOM, LICENSE and release notes. Self-signed code-signing and TLS certificates are for internal testing and will later be replaced with organization or publicly trusted certificates.

## License

Winception is released under **AGPL-3.0-only**. When users interact with a modified version over a network, the corresponding source must be offered. Third-party components retain their own licenses; see packaged `THIRD-PARTY-NOTICES.md` and the SBOM.

## Support and security

Supported hosts are Windows 11 Pro and Enterprise x64. Report vulnerabilities privately as described in repository `SECURITY.md`; general support scope is in `SUPPORT.md`. Public release still requires upstream redistribution review, an organization/public code-signing certificate, and multi-client plus physical-laptop acceptance.
